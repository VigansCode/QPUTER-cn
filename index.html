# 项目背景
 
零号是深度求索实验室R1推理引擎的第一个完整实现。
这个构建整合了我们在思维链推理(CoT)和强化学习方面
的最新突破。不同于DeepSeek-V3的生产版本，零号专注
于深层推理能力的极限探索。
 
本系统采用混合专家架构(MoE)，配备256个专家节点，
支持最多128层深度推理。训练数据包含15T tokens的
多语言语料，重点强化中文和代码理解能力。
 
# 技术规格
 
• 模型参数: 671B (MoE, 37B activated per token)
• 推理延迟: 平均 2.8ms (A100 80GB)
• 上下文窗口: 128K tokens
• 训练框架: HAI-LLM (自研)
• 部署位置: 杭州余杭数据中心 - 实验集群B
